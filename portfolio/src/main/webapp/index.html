Mini-C Compiler

  • Implemented a compiler for a subset of C language that supports variable declaration, arithmetic, relational and logical expressions, function calls, conditional expressions, and loops.
  • Used Flex for lexical analysis, Bison for syntax and semantic analysis and code generation. Generated 3-address code in Quadruple format in the first pass and MIPS code in the second pass.

L2-Hybrid Cache (Research Project)

  • Designed an algorithm using hybrid STT-RAM with variable retention time to optimize hit rate and improve latency in the L2 cache.
  • Implemented the algorithm on the gem5 simulator. It showed an average of 10% CPI improvement over standard STT-RAM.

CSEA Website

  • Lead a team of five students to develop the official website for the CSEA(Computer Science and Engineering Association) with main features including personalized alumni dashboard, events information, and event calendar.
  • Used Django to develop backend, SQLite3 as Database and HTML, CSS, Javascript for frontend.

Socket Programming

  • Developed a Base64 Encoding Communication Protocol between server and client to over TCP connection. Implemented a concurrent server to accept connections from multiple clients and serve all of them concurrently.
  • The client and server interact using three types of messages: Type 1(Encoded text message), Type 2(ACK message), Type 3(Connection Termination message).

Speech Laboratory Portal

  • Developed a website and Android application to manage all activities of Speech Laboratory of IIT Guwahati including assignment submissions, course creation and registration, event calendar, discussion rooms, Q/A sessions, and attendance information.
  • Used Android Studio to develop app, Django to develop backend, Google Firebase as Database and HTML, CSS, Javascript for frontend.

Pint OS

Implemented the following for an instructional operating system framework - PintOS.
  • Threads Module: Multi-level feedback queue scheduler and synchronization primitives.
  • User Programs: System call handler and system calls (exec, wait, read, write, etc).
  • Virtual Memory Management: Paging, stack growth, swapping, and page replacement.

Automated Item Dispenser

  • The Automated Item Dispenser streamlines the process of issuing weekly lab equipment for all involved. It allows for efficient weekly lab item dispensing for students and easy restocking of machines for instructors.
  • Sensors and Motors: Stepper Motor, Ultrasonic Sensor, Servo Motor, Colour Sensor, RFID Scanner, Matrix Keypad. To communicate user and system error, we use MQTT and SMTP protocol.

Movie Review Generator

  • Designing a deep-learning model which uses sentiment analysis to auto-complete movie reviews based on its synopsis and user rating. The model applies character embeddings(one-hot encoding)to learn the sentiment of reviews.
  • The dataset consists of a self-generated corpus of around 1 million user reviews from IMDB. Used LSTM with Keras library for text generation model, Beautifulsoup and Selenium for scraping.

Election Portal

  • Developed an Election Management system in Visual C++. It aims to organize and manage elections at all levels within IIT Guwahati (Hostels, Clubs, Student Council, etc.).
  • The software maintains the complete anonymity of the voter and ensures a secure election. The software provides a detailed analysis of the percentage votes for each candidate and voters' turnout.
